# Sensus project

> Sensus is a social media project made by [me](https://github.com/vbzt)

## How to use
> Installation

```bash
npm install
```

> Running the app

 - Create a .env file that contains:
	- POSTGRES_HOST
	- POSTGRES_DATABASE
	- POSTGRES_USER
	- POSTGRES_PASSWORD
- Start the server
```bash
npm run start
```

## Technologies

 1. Node.js
 2. Express
 3. Sequelize
 4. PostgreSQL
 
> Other dependencies:

 1. cookies/sessions
 2. flash messages
 3. bcrypt
 4. dotenv
